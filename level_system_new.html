Performance Marketer

Performance Marketing

공통역량 (Common Competency)

리더십

  1. 업무 수행 시 세밀한 지도가 필요함
  2. 리더의 지도를 따라 업무를 수행할 수 있음
  3. 독립적으로 모든 업무를 수행 가능함
  4. 기본적인 업무 코칭 가능
  5. 동료의 문제를 효과적으로 해결할 수 있음
  6. 소규모 팀(2명)을 리드할 수 있음
  7. 중간 규모 팀(3~4명)을 리드할 수 있음
  8. 대규모 그룹을 리드할 수 있음
  9. 회사의 주요 리드 역할에 참여 가능함

커뮤니케이션

  1. 자신의 생각을 편하게 전달할 수 있음
  2. 클리어하게 자신의 의견을 전달할 수 있음
  3. 논리적인 커뮤니케이션 가능함
  4. 정서적 공감을 만들어내는 커뮤니케이션 가능함
  5. 소규모 그룹 앞에서 효과적으로 소통할 수 있음
  6. 대중 앞에서 커뮤니케이션 가능함

협업능력

  1. 오픈 커뮤니케이션에 문제 없음
  2. 동료의 동기부여에 부정적 영향을 미치지 않음
  3. 업무적인 협업이 가능함

직무역량 Performance Marketer

프로젝트 관리 능력

  1. 프로젝트 운영을 서포트할 수 있음
  2. 고난도 프로젝트를 제외한 운영이 가능함
  3. 단일의 소규모 광고주를 관리할 수 있음
  4. 다수의 소규모 광고주를 관리할 수 있음
  5. 중요 광고주를 관리할 수 있음
  6. 챌린지한 프로젝트를 운영할 수 있음
  7. 리스크를 예측하고 효과적으로 관리할 수 있음

마케팅 전략 기획

  1. 마케팅 원하는 주요 지표를 이해하고 활용할 수 있음
  2. 광고 기획이 가능함

매체에 대한 이해도

  1. 주요 매체의 기본 세팅이 가능함
